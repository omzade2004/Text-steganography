# Text Steganography Web Application

A browser-based tool to secretly encode and decode hidden messages within plain text using steganographic techniques.

## Features
- Encode a secret message inside normal-looking text
- Decode hidden messages instantly
- 100% client-side — no data is sent to any server
- Zero-storage architecture for complete user privacy

## Tech Stack
- HTML5
- CSS3
- JavaScript 

## How to Use
1. Clone the repo
   git clone https://github.com/omzade2004/text-steganography
2. Open index.html in any browser
3. Type your message, encode it, and share!
